AutoWeb Setup
Guide to setup listings

Setting up the LAS code

1. Copy and paste the code in Box 1 before the closing body tad </body> on your results page.

<script
aw-placement= "131"
src= "//cdn.awadserver.com/widget/js/awloader.min.js">
</script>

2. Copy and paste the code in Box 2 on the section where you want to display our ads.

<div class= "awlistings" aw-implement="348" ></div>

Autoweb Parameter Definitions

Brief description of our widget parameters

Name Description Parameter Example
* aw-implement Specific Implementation ID provided by AutoWeb aw-implement="999"
* aw-placement Your placement ID provided by AutoWeb aw-placement="999"
* src URL for Autoweb's widget functionality src="//cdn.awadserver.com/widget/js/aw-loades.js"
aw-cta Set your costum call to action here.
You can use html tags and the following tags in order
to display additional info:
{Make},{Model},{cityName},{stateName},{ZipCode}.		 aw-cta="<h3>Find your new {Make},{Model} on {cityName},{stateName} / {ZipCode}</h3>"
aw-display-count Specific the number of ads to show aw-display-count="5"
Important Note:Do not modify it unless it is requested.

Query String parameters

Additionally you can pass the make / model / zip code on the query string, using the following parameters:
*Make
*Model
*Zipcode
